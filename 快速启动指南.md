demo6-cloud 快速启动指南

前置条件

- JDK 17+
- Maven 3.8+
- MySQL 8.x (已安装)
- Redis 7.x (需安装)



方式一：完整微服务模式（使用 Nacos）⭐ 推荐

1. 启动 Nacos

```bash
# 解压 nacos-server-2.4.3.zip
# 进入 nacos/bin 目录

# Windows:
startup.cmd -m standalone

# Linux/Mac:
sh startup.sh -m standalone

# 访问 Nacos 控制台: http://localhost:8848/nacos
# 默认账号密码: nacos/nacos
```

### 2. 启动 Redis

```bash
# Windows (如果已安装):
redis-server

# 或使用 Docker:
docker run -d -p 6379:6379 --name redis redis:7-alpine
```

### 3. 启动微服务（按顺序）

```bash
# 先编译
mvn clean package -DskipTests

# 终端1 - 启动 Gateway (8080)
cd demo6-gateway && mvn spring-boot:run

# 终端2 - 启动用户服务 (8081)
cd demo6-user-service && mvn spring-boot:run

# 终端3 - 启动账号服务 (8082)
cd demo6-account-service && mvn spring-boot:run

# 终端4 - 启动订单服务 (8083)
cd demo6-order-service && mvn spring-boot:run

# 终端5 - 启动还价服务 (8084)
cd demo6-bargain-service && mvn spring-boot:run
```

### 4. 访问系统

- 用户端: http://localhost:8080/index.html
- 管理端: http://localhost:8080/game-account-manager.html
- 注册页: http://localhost:8080/register.html
- Nacos 控制台: http://localhost:8848/nacos

## 方式二：快速开发模式（无需 Nacos）

修改 Gateway 配置，将 `lb://` 改为直接 URL：

1. **修改 `demo6-gateway/src/main/resources/application.yml`**
   - 将所有 `uri: lb://user-service` → `uri: http://localhost:8081`
   - `uri: lb://account-service` → `uri: http://localhost:8082`
   - `uri: lb://order-service` → `uri: http://localhost:8083`
   - `uri: lb://bargain-service` → `uri: http://localhost:8084`

2. **各服务禁用 Nacos** — 在 `application.yml` 中添加：
```yaml
spring:
  cloud:
    nacos:
      discovery:
        enabled: false
      config:
        enabled: false
```

3. **Gateway 添加 `@EnableDiscoveryClient` 去掉** 或设置：
```yaml
spring:
  cloud:
    discovery:
      enabled: false
```
